# 🎵 Sistema de Toca-Discos em Java

Desafio desenvolvido em aula para praticar orientação a objetos em Java. O sistema simula um toca-discos com controle de CD, energia e reprodução.

## 📦 Classes

- **CD** — representa um disco de música com título, artista, número de faixas e duração
- **TocaDisco** — representa o aparelho com controle de ligar/desligar, inserir/ejetar CD e play/pause/stop
- **EstanteCds** — representa uma estante que armazena uma coleção de CDs, permitindo guardar e listar os discos
- **Main** — classe de teste que simula o fluxo completo de uso

## 🔧 Tecnologias

- Java

## ▶️ Como rodar

1. Clone o repositório
2. Compile as classes com `javac *.java`
3. Execute com `java Main`
